Cells

Cells is an application for fluid artists by fluid artists. It stores an artists paintings and the components used to make up that painting, so that they can return at any time to see what made up a previous painting and try to replicate parts they liked or another artist can take a look and find inspiration for themselves!


Technologies used: React.js, Node.js, Express.js, PostgresSQL, SWAL and Amazon Web Services S3 Cloud
